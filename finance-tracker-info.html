The Finance Tracker is a Vite React project that must be served by a dev or static server. Opening its index.html directly in the browser will show a blank page because the app uses ES module imports and absolute paths.

To run locally (recommended)

Open a terminal in the finance-tracker folder and run:

npm install
npm run dev

To build a static version

Run:

npm run build
npm run preview

If you'd like, you can open the raw project files (may not run) or return to the landing page.
